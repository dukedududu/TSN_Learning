# TSN Learning Guides

This repository contains learning guides and documentation for Time-Sensitive Networking (TSN).

## Contents

*   **[frer_guide.html](frer_guide.html)**: Guide on Frame Replication and Elimination for Reliability (FRER).
*   **[ptp_guide.html](ptp_guide.html)**: Guide on Precision Time Protocol (PTP).
*   **[tsn_config_guide.html](tsn_config_guide.html)**: Guide on configuring TSN parameters and settings.
*   **[tsn_control.html](tsn_control.html)**: Information about TSN control plane and mechanisms.
*   **[tsn_step_guide.html](tsn_step_guide.html)**: Step-by-step tutorial for TSN implementation or experiments.

## Usage

You can view these guides directly by downloading the HTML files and opening them in your web browser.
